# ASCII Blackjack 21

My Final Project for Stanford's **Code in Place 2026** course. This is a minimalist, terminal-based Blackjack game built in Python using Object-Oriented Programming (OOP).

## Features
* **Horizontal ASCII Cards:** Custom visual rendering of cards in the terminal.
* **Smart Ace Logic:** Automatic score adjustment (11 to 1) to prevent busting.
* **Bankroll System:** Interactive betting mechanism with persistent chip counts.
* **Session Stats:** Tracks your overall Wins, Losses, and Ties across rounds.

## How to Run
```bash
python main.py
